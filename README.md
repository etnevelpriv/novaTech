Készíts egy egyszerű weboldalt egy szabadon választott témában!
A tartalom lehet statikus, pl.
Cég bemutatkozó oldala
Blog főoldala
Webshop ajánlott termékei
stb.

Készítsd el az oldal megjelenését az alábbiak szerint:
CSS keretrendszer NEM használható! Saját CSS-t kell írni.
Az oldal álljon több szekcióból. A tetején legyen egy navigációs sáv, amivel az oldalon belül lehet navigálni
Ha egy menüelem fölé visszük az egeret, áttűnéssel változzon meg a stílusa!
Legalább egy szekciót lehessen animálva kinyitni / összecsukni!
(pl. a magassága / szélessége / betűmérete / áttetszősége stb. folyamatosan legyen kisebb v. nagyobb)
Ehhez egy kis programkódra is szükség lehet.
A főbb színeket, háttérképet (ha van), méreteket CSS változók segítségével definiáld.
Az oldal ismerje fel, hogy a felhasználó sötét v. világos módot használ, a CSS változóidat is eszerint állítsd be
Ha a felhasználó op. rendszer szinten kikapcsolja az animációkat, akkor ezt a weboldal is tükrözze
A megvalósításban legyen reszponzív elem, min. 1 törésponttal.
